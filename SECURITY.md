# Security Policy

## Supported Versions

Only the latest main branch is currently supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you find a security vulnerability, do not open a public issue. 

Please report it through one of the following secure channels:
* **Encrypted Communication:** Submit details via PGP key or secure anonymous drop.
* **Response Time:** Vulnerabilities will be assessed within 48 hours.

> "Confidence is a weakness. Always assume the system is already breached."
